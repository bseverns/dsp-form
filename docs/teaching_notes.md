# Teaching Notes

## Student-friendly framing

Sound can become shape in more than one way.

Students do not need to understand all of DSP before they can make meaningful decisions. They need a few reliable mappings:

- Louder can become taller.
- Brighter can become sharper or higher.
- Beats can become ribs.
- Repetition can become pattern.
- A seed can make a family.

## Starter activity

1. Choose a short sound.
2. Generate a terrain tile.
3. Change only one parameter.
4. Generate a second tile.
5. Compare them.
6. Explain the difference using one sentence and one drawing.

## Strong reflection prompts

- What part of the sound is easiest to see?
- What part disappeared?
- What did the computer decide?
- What did you decide?
- What would you change if this had to be held in a hand?
- What would you change if this had to be projected on a wall?

## Good constraint

Every student export gets a manifest. The manifest is the label on the specimen jar.

# Concept

DSP Form Lab treats sound and control data as material for sculpture.

This is not only audio visualization. The goal is to build a repeatable instrument that turns signal behavior into bodies:

```text
audio/control source → features → geometry grammar → mesh → manifest → print/render/archive
```

## Working claim

A generated object is strongest when it is both:

1. **Expressive** — it feels like the signal had a body hidden inside it.
2. **Accountable** — its source, seed, parameters, generator, and export path are documented.

## A practical aesthetic

Useful language for this project:

- **Signal fossil** — sound as something that left a trace in material.
- **Waveform body** — amplitude and spectral motion as ribs, tendons, folds, skins.
- **Control sculpture** — knobs, MIDI CCs, OSC messages, and envelope followers as decisions in space.
- **Procedural reliquary** — a small object that contains a performance, not by recording it, but by hardening its behavior.

## What should be preserved

Every export should preserve:

- Source media path or pointer.
- Analysis settings.
- Geometry grammar.
- Generator version.
- Random seed.
- Dimensional scale.
- Mesh warnings.
- Output files.

The manifest is not paperwork after the art. It is part of the instrument.

# Example 03 — Circular Vessel

Wrap time around a circle.

```bash
dspform vessel audio/samples/sine_sweep.wav \
  --out outputs/obj/sine_sweep_vessel.obj \
  --height 90 \
  --base-radius 20 \
  --radial-amp 12 \
  --layers 48 \
  --ridge-amp 3
```

Reflection:

- Does it feel like a container, relic, tower, or instrument part?
- What does a beat become when it is wrapped around a circle?
